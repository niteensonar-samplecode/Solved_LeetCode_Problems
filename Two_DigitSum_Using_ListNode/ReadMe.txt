Two digit sum using ListNode:
Given a ListNode in the reverse of the order of the digits, return the sum of the two numbers represented by the ListNode. 
The ListNode will have a value of 0-9 and a next pointer to the next digit in the number. 
The output should be a ListNode representing the sum of the two numbers.
Return the sum as a ListNode.

 

Example 1:


Input:  l1 = 2 -> 4 -> 3 (342)
Input:  l2 = 5 -> 6 -> 4 (465)
Output: 7 0 8 
Explanation: "aba" is also a valid answer.


Example 2:


Input: s = "cbbd"
Output: "bb"
